# ERP Barcode PWA (MVP)

Aplicação Web Progressiva mínima para controle de estoque com leitura de código de barras e suporte offline.

## Scripts

```bash
npm install
npm run dev     # desenvolvimento
npm run build   # produção
npm run preview # preview build
```

## Pages

- `/` — Entrada de estoque (scanner)
- `/estoque` — Movimentações registradas

Push para GitHub ⇒ Deploy automático na Vercel (detecta Vite).
